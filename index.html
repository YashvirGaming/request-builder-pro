Request Builder PRO

Universal HTTP request analyzer → cyberpunk Python checker generator. Paste any request from Chrome DevTools, HTTPDebugger, Burp Suite or cURL and get a full working multi-threaded checker in one click.

Python 3.10+ CustomTkinter Multi-threaded Proxy Support MIT License

Features

5 Input Parsers

HAR, Burp Suite, HTTPDebugger XML, cURL, and raw HTTP — all auto-detected

Smart Code Gen

Detects email/pass fields, payload type, CSRF tokens — generates clean valid Python

Checker GUI Output

Generated script is a full cyberpunk CTk checker with combos, proxies, threads, stats

Session Chaining

Auto-extracts CSRF tokens from login pages and injects them into POST requests

Proxy Rotation

ip:port and ip:port:user:pass formats, round-robin rotation per thread

Syntax Highlighting

Generated code box uses a token-level Python highlighter — Dracula-style neon palette

Generated checker features

  • Cyberpunk dark CTk GUI — #050508 black, #00fff7 neon, #ff2d78 pink, #7b2fff purple
  • Animated pulse indicator — cycles frames while checker is running
  • Combo loader — email:pass format, skips malformed lines
  • Proxy loader — ip:port or ip:port:user:pass, round-robin per thread
  • Thread slider — 1 to 200 threads, live label
  • Stats board — Total / Hits / Bad / Retries / Errors / CPM
  • Progress bar with done / total counter
  • Output log — colour-coded: cyan=hit, red=bad, yellow=retry, pink=error
  • Hits written to timestamped hits_YYYYMMDD_HHMMSS.txt immediately on match
  • CSRF session chaining — auto-fetches token, injects into POST
  • Retry logic — reads Retry-After on 429, sleeps on 5xx, custom keyword triggers
  • All 6 parse modes — json, lr, regex, css, xpath, jsonpath
  • Credits bar — 10 social links at the bottom of every generated window
